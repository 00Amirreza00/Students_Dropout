# 🎓 Students' Dropout and Academic Success

This project provides a comprehensive data analysis and machine learning framework to predict students' dropout rates and academic success based on socio-economic, demographic, and academic performance data. The project utilizes advanced resampling techniques to handle class imbalance, a common challenge in educational datasets.

## ✨ Features
- **Exploratory Data Analysis (EDA):** Interactive visualization of class distributions and data statistics.
- **Advanced Resampling:** Implements multiple techniques to tackle class imbalance:
    - `SMOTE` (Synthetic Minority Over-sampling Technique)
    - `Random Over Sampling` (ROS)
    - `Random Under Sampling` (RUS)
    - `ADASYN` (Adaptive Synthetic Sampling)
- **Model Comparison:** Evaluates performance across `Random Forest Classifier` and `K-Nearest Neighbors` (KNN).
- **Metric Focused:** Prioritizes Macro-averaged F1 Score to ensure fairness across imbalanced classes.

## 🛠 Tech Stack
- **Python:** Core programming language.
- **Streamlit:** Used for the interactive web dashboard.
- **Scikit-learn:** Pipeline management, preprocessing, and classification models.
- **Imbalanced-learn:** Resampling strategies.
- **Pandas/Numpy/Matplotlib:** Data manipulation and visualization.

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/00Amirreza00/Students_Dropout
   cd Students_Dropout
   ```
2. Install dependencies (ensure you have `pip` installed):
   ```bash
   pip install pandas numpy matplotlib streamlit scikit-learn imbalanced-learn
   ```

## 💡 Usage

To run the interactive analysis dashboard, execute the following command in your terminal:

```bash
streamlit run Students_Dropout_Str.py
```

This will launch a local web server (usually at `http://localhost:8501`) where you can explore the data, compare models, and view the confusion matrices generated by different sampling techniques.

## 📊 Conclusion
Based on the analysis, the **Random Forest Classifier (RFC)** combined with **Random Oversampling** provides the most robust performance for predicting academic outcomes in this specific dataset.
